# Toast
js toast plugins base on jQuery


Usage
======
```javascript
<script>
  toast({
    msg: '文字内容'，
    position: 'center',
    //position: 'top 100'     弹窗距离顶部100px 
    //position: 'bottom 100'  弹窗距离底部100px
    duration: '3000'          //单位ms
  });
</script>
```
